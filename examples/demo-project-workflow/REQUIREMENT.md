# REQUIREMENT

## Original Request

Add priority labels to issues.

## Confirmed Goal

Users can assign low, medium or high priority to each issue and filter the issue list by priority.

## Edge Cases

- Existing issues without priority default to medium.
- Unknown priority values are rejected by the API.

## Acceptance Criteria

- [x] Create, update and list APIs expose priority.
- [x] UI shows priority and supports filtering.
- [x] Tests cover default priority and invalid priority.
